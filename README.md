# SGTR Mitigates and Reverses EM — Interactive Chart

Interactive bar chart visualization for embedding in Notion.

## Deploy to GitHub Pages

1. Create a new repo on GitHub
2. Push this code to `main`
3. Go to **Settings → Pages → Source** → select **GitHub Actions**
4. Site deploys to `https://USERNAME.github.io/REPO_NAME/`

## Embed in Notion

Use `/embed` block in Notion and paste the GitHub Pages URL.
